# 认知框架驱动案例生产模板 v1.0

## 核心原则

**不手写案例场景。** 用已有的认知框架去解剖生活原型。框架自带三刀和追问。

```
生活原型             认知框架                         案例
   ↓                    ↓                               ↓
"我该不该跳槽"  → 认知主权的三层穿透  → 意义感知×时间折叠刀
                 → 决策框架的四步拆分  → 价值判断×换位刀
                 → 锋刃标准的自我攻击  → 世界建构×反事实刀
```

## 五个认知框架 → 案例映射

### 框架一：认知主权（三层穿透）

**来源：** cognitive-sovereignty — AI平权了信息处理能力后，个体不可替代性在意义感知、价值判断、世界建构。

**映射到案例：**

| 认知主权的维度 | 对应三刀 | 追问方向 |
|:--------------|:---------|:---------|
| 意义感知——你在做的事还能感知到"成长"吗？ | 时间折叠刀 | 三年前的新鲜感→今天的疲态→三年后的麻木 |
| 价值判断——什么才是你的"价值"？稳定还是可能性？ | 换位刀 | 不同角色对"价值"的定义冲突 |
| 世界建构——你在建哪种人生版本？ | 反事实刀 | 打破"现在安全"的假设 |

**适用原型：** 安全vs成长（跳槽、换专业、离婚、退休）

---

### 框架二：决策框架（四步决策法）

**来源：** decision-framework — 定义决策→拆选项→三维评估→压力测试

**映射到案例：**

| 决策框架的步骤 | 对应三刀 | 追问方向 |
|:-------------|:---------|:---------|
| 定义决策——你真正在决定什么？ | 时间折叠刀 | 表面选项下藏着的真正问题 |
| 拆选项——除了A和B还有C吗？ | 换位刀 | 每个角色的立场下还藏着第三选择 |
| 三维评估（效果×风险×成本） | 反事实刀 | 每个选项的核心假设 |

**适用原型：** 人情vs利害（借钱、开口不说、站队）

---

### 框架三：第一性原理穿透

**来源：** first-principles — 向下追问直至触及最根本的约束条件和底层逻辑

**映射到案例：**

| 第一性原理的追问 | 对应三刀 | 追问方向 |
|:---------------|:---------|:---------|
| 这件事最底层的约束是什么？ | 反事实刀 | 剥掉所有"我觉得""别人说"，剩下什么？ |
| 如果从头开始，你会怎么选？ | 时间折叠刀 | 回到出发点重新审视 |
| 这个需求的本质是什么？ | 换位刀 | 不同角色看到的"需求"是什么 |

**适用原型：** 需要vs想要（双十一、买车、买房）

---

### 框架四：锋刃标准（停顿感检验）

**来源：** sharp-edge-standard — 关键论述须能带来"停顿感"，避免正确的废话

**映射到案例：**

| 锋刃标准的维度 | 对应三刀 | 追问方向 |
|:-------------|:---------|:---------|
| 独特性——这个判断是真的还是随大流？ | 换位刀 | 如果周围人都在做相反的选择，你还会这么选吗？ |
| 穿透性——这个判断打透了问题的本质吗？ | 反事实刀 | 你绕开了表面症状，还是只换了种说法描述症状？ |
| 冲击力——这个判断让你自己停顿了吗？ | 时间折叠刀 | 一年后回头看，这个判断会让你觉得自己聪明还是后悔？ |

**适用原型：** 花钱买安心（报班、买课、囤书）

---

## 生产流程

```
第一步：识别原型
  → 这个决策属于哪种原型困境？
  → 从四个框架中选一个主框架

第二步：框架投射
  → 把认知框架的三个维度分别投射到三把刀上
  → 每把刀自动生成3个追问方向

第三步：具体化
  → 把抽象的框架语言翻译成你日常能遇到的具体场景
  → 举例：不说"意义感知在凋亡"，说"每天早上起床不想去上班"

第四步：追问填充
  → 从苏格拉底五类追问中选2-3个
  → 每个追问必须扎在一个具体的盲区上
```
